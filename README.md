# ✨ **Touchless Tourism Kiosk**

A **gesture-controlled, touchless tourism kiosk system** built using **Computer Vision and AI**. This project enables users to explore **tourist destinations without physical interaction**, ensuring a **safe and interactive experience**.

---

## 🚀 **Features**

### ✋ **Hand Gesture Navigation**

* **1 Finger** → Pointer movement
* **2 Fingers** → Select option
* **3 Fingers** → Open menu
* **4/5 Fingers** → Go back / Reset

### 🧠 **Real-Time Hand Tracking**

* Implemented using **MediaPipe Hand Landmarker**
* Smooth pointer tracking with **gesture stabilization**

### 🖼️ **Interactive Tourism Interface**

* Dynamic menu of **tourist places**
* Background changes based on **selection**

### 📍 **Detailed Place Information**

* **Description**
* **Location**
* **Timings**
* **Entry details**

### 🔗 **QR Code Integration**

* Scan QR to open **Google Maps location**

### 🎯 **Smooth UI Effects**

* **Pointer smoothing**
* **Gesture delay handling**
* Visual feedback with **animations**

---

## 🛠️ **Technologies Used**

* **Python**
* **OpenCV** – Image processing & UI rendering
* **MediaPipe** – Hand gesture detection
* **NumPy** – Numerical operations
* **QR Code (qrcode)** – Generate scannable map links

## ⚙️ **Installation & Setup**

### 1️⃣ **Clone the repository**

```bash
git clone https://github.com/adithyaps-sudo/Touchless-kiosk.git
cd Touchless-kiosk
```

### 2️⃣ **Install dependencies**

```bash
pip install -r requirements.txt
```

### 3️⃣ **Run the project**

```bash
python main.py
```

## 🎮 **How to Use**

1. Start the application
2. Use **hand gestures** in front of the camera
3. Navigate through the menu using gestures
4. Select a tourist place
5. View details and scan QR for location
